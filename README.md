# actuellement-TAL
> *Chaque linguiste renvoyé, un humain ordinaire de plus qui fait du TAL.*  
> (Every linguist fired, one more ordinary human joins NLP.)

Putting the "Natural" back in Natural Language Processing by removing the "Processing" and keeping only the "Natural" :)

**TAL** stands for ***Traitement Automatique du Langage*** (Automatic Language Processing) in French. 
However, due to the author's incomplete command of French orthography, **TAL** is reinterpreted here as **Totally Amateur Linguistics** (according to the developing philosophy of the experiments), or alternatively, **Automatic Language Treatment** — emphasizing how I automatically *treat* the language, or considering that some linguists clearly require *treatment* (intervention) rather than computational analysis.

*(For English readers: If your project takes language as input and does something with it — welcome. No gatekeeping here.)*

## 给中文读者看的部分
### 仓库名称：自然（的）语言处理
**Q：这到底是什么玩意？**
**A：只要你的输入是“语言”，输出是对语言的某种“处理”，就算。**
包括但不限于以下内容：
- 用脚本把文本里所有的“的”换成“の”（这个真的做了，就是那个`the_first_script.py`）
- 训练一个识别微博评论是不是在阴阳怪气的模型 
- 计算你和朋友的聊天记录里出现了多少次“基本上”（这叫语料统计）
- 用AI生成100篇水课论文，然后分析“首先其次再次”出现了几次（这叫话语分析，正经研究方向）  
总之，你觉得算NLP，那就是NLP。这里不设门槛。
（如果你不确定——那正好，这个仓库就是为“这也算NLP？”准备的。）
